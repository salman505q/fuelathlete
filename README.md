# ⚡ FuelAthlete - PWA

## الملفات المطلوبة
```
index.html       ← التطبيق الرئيسي
manifest.json    ← إعدادات PWA
sw.js            ← Service Worker (للعمل Offline)
icon-192.png     ← أيقونة التطبيق
icon-512.png     ← أيقونة كبيرة
```

## نشر مجاني على Vercel (5 دقائق)

### الطريقة السهلة (بدون كود)
1. روح https://vercel.com واعمل حساب مجاني
2. اسحب مجلد الملفات وأفلته في الموقع
3. خلاص! راح تحصل رابط مثل: https://fuelathlete.vercel.app

### أو عبر GitHub
1. ارفع الملفات على GitHub repo
2. في Vercel: New Project → Import من GitHub
3. Deploy ✅

## كيف يضيفه المستخدم للشاشة الرئيسية؟

### آيفون (iOS)
1. افتح الموقع في Safari
2. اضغط زر المشاركة 📤
3. اختر "Add to Home Screen"
4. اضغط Add ✅

### أندرويد
- Chrome راح يطلع banner تلقائي "Install App"
- أو: القائمة ← "Add to Home Screen"
